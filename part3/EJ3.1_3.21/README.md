# Phonebook Backend

Backend for the Phonebook application (Full Stack Open Part 3).

## 🌍 Live API

https://fullstackopen-2026-fhfq.onrender.com

## 📌 Endpoints

- GET /api/persons
- GET /api/persons/:id
- POST /api/persons
- DELETE /api/persons/:id

## 🚀 Tech Stack

- Node.js
- Express
- Morgan
- CORS

## 🛠️ Running locally

npm install
npm start
